Create Your First Schema

What are schemas?

Schemas are JSON Schema definitions that specify the structure and validation rules for your extracted data. When you link a schema to a prompt, DocRouter.AI ensures the AI output matches your defined structure.

Example: Invoice Schema

{
  "type": "object",
  "properties": {
    "invoice_number": {"type": "string"},
    "date": {"type": "string"},
    "vendor": {"type": "string"},
    "total": {"type": "number"}
  },
  "required": ["invoice_number", "date", "total"]
}
