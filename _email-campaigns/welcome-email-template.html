Smart Document Router is an open-source Document Understanding and Data Extraction tool that uses AI and LLMs to extract structured data from unstructured documents.

Getting Started

1. Upload Documents - Upload PDFs, images, or documents from faxes, email, and ERPs.

2. Create Prompts & Schemas - Define what data to extract and structure the output. Use tags to route documents automatically.

3. Review & Verify - Use interactive forms to review AI-extracted data with human-in-the-loop accuracy.

4. Export & Integrate - Export structured data or integrate using Python/TypeScript SDKs or REST API.
